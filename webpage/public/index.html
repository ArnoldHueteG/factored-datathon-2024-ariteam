Welcome to AriTeam
Factored's Datathon 2024

The 2024 Factored Datathon was an exhilarating event that brought together 352 participants from around the world. Our mission was to extract crucial insights from global events, enabling early intervention and strategic planning to help businesses and organizations navigate complex and volatile environments. We delved deeply into the extensive and dynamic dataset provided by the Global Database of Events, Language, and Tone (GDELT).

Project Architecture

Leveraging Google Cloud and Open Source Technologies to Deliver a Powerful Data Solution

  • Version Control and Continuous Integration/Continuous Deployment (CI/CD) with GitHub Actions

  • Container Orchestration using Docker with Artifact Registry for image management

  • Data Processing and Analytics: ETL Pipelines with Dataflow, advanced analytics using BigQuery

  • Knowledge Graph Implementation: Neo4j database powering core application logic

  • Data Visualization and Insights: Interactive dashboards with Streamlit, business intelligence reporting via Looker

  • Web Hosting and Deployment: Firebase Hosting for efficient and scalable web application serving
